현재 플랜에서 다음 작업 단계를 찾아 실행합니다.

## 1. 플랜 파일 찾기

`.claude/plans/` 폴더에서 플랜 파일을 읽으세요.
파일이 여러 개면 가장 최근에 수정된 것을 사용합니다.

## 2. 진행 상황 파악

아래 두 가지를 읽어서 어디까지 완료됐는지 파악하세요.

```
git log --oneline -20
```

플랜의 각 단계와 커밋 메시지를 비교해서 완료된 단계를 식별합니다.
커밋이 없거나 매칭되는 커밋이 없으면 첫 번째 단계부터 시작합니다.

## 3. 다음 단계 선언

작업 시작 전 반드시 아래 형식으로 먼저 출력하세요:

```
✅ 완료된 단계: [완료된 단계 목록 또는 "없음"]
▶️ 이번에 할 단계: [단계명]
📋 내용: [이번 단계에서 할 작업 간략 설명]
```

## 4. 작업 실행

해당 단계의 작업을 수행합니다.
플랜에 명시된 내용대로 구현하되, 현재 코드 상태를 먼저 읽고 시작하세요.

## 5. 커밋

작업이 완료되면 변경 파일을 커밋합니다.
커밋 메시지는 플랜의 단계명이 명확히 포함되도록 작성합니다.
예: `feat: [플랜 단계명] — 구체적인 작업 내용`

## 6. 다음 단계 안내

커밋 완료 후 아래를 출력하세요:

```
🎉 [단계명] 완료. 커밋됨.

남은 단계:
- [ ] 다음 단계명
- [ ] 그 다음 단계명
...

/clear 후 /next-step 으로 계속하세요.
```

모든 단계가 완료됐으면 `/done` 을 실행하세요.
