# TastingNote 기능 목록 (초안)

> 친구와 함께 구상한 초안입니다. 완성본 아님 — 세션마다 업데이트 예정.
> 우선순위: [필수] → [UX] → [추가] → [보류]

---

## 1. 인증 / 로그인

- [ ]  [필수] 회원가입 (이메일, 비밀번호, 닉네임)
- [ ]  [필수] 로그인 / 로그아웃
- [ ]  [필수] 비밀번호 마스킹 + 해제 토글
- [ ]  [UX] 로그인 실패 원인 구분 메시지 (이메일 없음 / 비밀번호 불일치 / 둘 다)
- [ ]  [UX] 비로그인 상태에서 좋아요 클릭 시 로그인 후 원래 페이지로 복귀
- [ ]  [추가] 비밀번호 찾기 / 재설정
- [ ]  [추가] 소셜 로그인 (카카오, 구글, 네이버)
- [ ]  [보류] 자동 로그인 (JWT 토큰 쿠키 저장)

## 2. 노트 작성

- [ ]  [필수] 노트 CRUD (작성 / 조회 / 수정 / 삭제)
- [ ]  [필수] 술 이름 자동완성
- [ ]  [필수] 태그 자동완성
- [ ]  [필수] 공개 / 비공개 설정
- [ ]  [UX] 사진 다중 업로드 (PC는 파일 선택, 모바일은 사진 앱)
- [ ]  [UX] 임시저장 기능
- [ ]  [UX] 별점 클릭 UI
- [ ]  [추가] 마신 날짜 직접 입력
- [ ]  [추가] 장소 입력
